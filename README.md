# AHA
<strong>Android Hybrid App</strong>
<p>V.0.1-BETA</p>

### What is this ?
AHA (Android Hybrid App) is a emulator full app for android.<br/>
You can use your CMS URL or your Responsive web URL like a Android app.

### Features
	- Splash Screen
	- News Screen
	- Intro Screen
	- Loading gif
	- OneSignal Push
	- Firebase
	- Gmail Login
	- Facebook Login
